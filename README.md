# La destinee-project
랜덤영상통화 및 소셜네트워크 서비스 <br><br>

## Getting Started ##
새로운 만남을 위한 공간이 필요할 때

실시간으로 새로운 만남을 제공하며, 마음에 드는 유저를 찾아보아요

더 많은 친구들을 만나 볼 수 있으며, 친구와 1:1 대화도 제공합니다.

La destinee 소셜 커넥트 서비스를 제공합니다.
